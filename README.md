**Customer Shopping Behavior Analysis**

This project demonstrates a complete Data Analytics workflow using Python, MySQL, and Power BI.

**Project Workflow**
1) Data Cleaning and Transformation
> Imported the raw dataset into Jupyter Notebook.
> Performed data cleaning, handling missing values, data transformation, and formatting using Python libraries.

2) Database Management
> Loaded the cleaned dataset into MySQL.
> Created and executed SQL queries to analyze customer behavior, purchasing patterns, discounts, revenue trends, and other business insights.

3) Data Analysis
> Explored the data using SQL to answer business-related questions.
> Generated meaningful insights from customer shopping activities.

4) Data Visualization
> Connected MySQL data to Power BI.
> Designed an interactive dashboard with charts and KPIs.
> Visualized key metrics such as sales trends, customer demographics, product performance, and purchasing behavior.

**Tools & Technologies Used**
> Python
> Jupyter Notebook
> Pandas
> MySQL
> SQL
> Power BI

**Key Outcomes**
> Cleaned and transformed raw data into an analysis-ready dataset.
> Extracted business insights using SQL queries.
> Built an interactive Power BI dashboard for effective data visualization and decision-making.
> Demonstrated an end-to-end data analytics workflow from data preparation to reporting.

**Skills Demonstrated**

> Exploratory Data Analysis (EDA)
